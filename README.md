# eCommerce By React
## Live: https://ecommerce-by-react.netlify.app/

### Features:
- Add To Cart And Remove from Cart and Adjust Quantities
- View Cart Products And Total Price
- Redux Toolkit (State Management)
- React Router
- Dark Mode
  
### Built With:
- React
- Tailwind
- Redux Toolkit
