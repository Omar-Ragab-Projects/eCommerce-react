# eCommerce By React
## Live: https://ecommerce-by-react.netlify.app/

### Features:
- Add to Cart and View Cart
- Remove from Cart and Adjust Quantities
- State Management with Redux Toolkit
- Navigation with React Router
- Dark Mode Toggle
- AOS (Animation On Scroll)
  
### Built With:
- React
- Tailwind
- Redux Toolkit
